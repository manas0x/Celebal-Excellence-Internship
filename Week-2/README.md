# Week 2 – E-Commerce Sales Database

This directory contains my submissions for the Week 2 SQL Assessment of the Celebal Excellence Internship. The task involved creating a simulated e-commerce database, inserting sample data, and writing various SQL queries to demonstrate data retrieval, filtering, aggregation, joins, and transactions.

## Files Included

- `db.sql`: Contains the DDL (Data Definition Language) to create the `ecommerce_sales` database, tables (`customers`, `products`, `orders`, `order_items`), and DML to insert sample data.
- `solutions.sql`: Contains the solutions to all 27 assignment questions organized by SQL concepts (Basics, Filtering, Aggregation, Joins, Advanced).
- `Week2_SQL.ipynb`: A comprehensive Jupyter Notebook containing the SQL queries alongside their execution outputs (screenshots included in the `images/` directory).
- `images/`: Directory containing screenshots of query outputs.
- `CEI_SQL_Task2_Week2.pdf` / `Celebal_SQL_Assignment.docx`: Original assignment problem statements.

## Setup Instructions

1. Execute `db.sql` in your SQL environment (e.g., MySQL Workbench) to build the database schema and insert initial data.
2. Review `solutions.sql` for query logic, or open `Week2_SQL.ipynb` to see the queries paired with their results.
